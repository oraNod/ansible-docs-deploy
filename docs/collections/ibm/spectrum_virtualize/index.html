Plugin Index

These are the plugins in the ibm.spectrum_virtualize collection:

Modules

  • ibm_sv_manage_awss3_cloudaccount module – This module configures and manages Amazon Simple Storage Service (Amazon S3) cloud account on IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_cloud_backups module – This module configures and manages cloud backups on IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_fc_partnership module – This module configures and manages Fibre Channel (FC) partnership on IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_fcportsetmember module – This module manages addition or removal of ports to or from the Fibre Channel(FC) portsets on IBM Spectrum Virtualize family storage systems.

  • ibm_sv_manage_ip_partnership module – This module manages IP partnerships on IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_provisioning_policy module – This module configures and manages provisioning policies on IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_replication_policy module – This module configures and manages replication policies on IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_snapshot module – This module manages snapshots (PiT image of a volume) on IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_snapshotpolicy module – This module manages snapshot policy configuration on IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_ssl_certificate module – This module exports existing system-signed certificate on to IBM Spectrum Virtualize family storage systems

  • ibm_sv_manage_truststore_for_replication module – This module manages certificate trust stores for replication on IBM Spectrum Virtualize family storage systems

  • ibm_sv_restore_cloud_backup module – This module restores the cloud backup on IBM Spectrum Virtualize family storage systems

  • ibm_sv_switch_replication_direction module – This module switches the replication direction on IBM Spectrum Virtualize family storage systems

  • ibm_svc_auth module – This module generates an authentication token for a user on IBM Spectrum Virtualize family storage system

  • ibm_svc_complete_initial_setup module – This module completes the initial setup configuration for LMC systems

  • ibm_svc_host module – This module manages hosts on IBM Spectrum Virtualize family storage systems

  • ibm_svc_hostcluster module – This module manages host cluster on IBM Spectrum Virtualize family storage systems

  • ibm_svc_info module – This module gathers various information from the IBM Spectrum Virtualize family storage systems

  • ibm_svc_initial_setup module – This module allows users to manage the initial setup configuration on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_callhome module – This module manages Call Home feature configuration on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_consistgrp_flashcopy module – This module manages FlashCopy consistency groups on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_cv module – This module manages the change volume for a given volume on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_flashcopy module – This module manages FlashCopy mappings on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_ip module – This module manages IP provisioning on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_migration module – This module manages volume migration between clusters on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_mirrored_volume module – This module manages mirrored volumes on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_ownershipgroup module – This module manages ownership group on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_portset module – This module manages portset configuration on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_replication module – This module manages remote copies (or rcrelationship) on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_replicationgroup module – This module manages remote copy consistency group on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_safeguarded_policy module – This module manages safeguarded policy configuration on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_sra module – This module manages remote support assistance configuration on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_user module – This module manages user on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_usergroup module – This module manages user group on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_volume module – This module manages standard volumes on IBM Spectrum Virtualize family storage systems

  • ibm_svc_manage_volumegroup module – This module manages volume groups on IBM Spectrum Virtualize family storage systems

  • ibm_svc_mdisk module – This module manages MDisks on IBM Spectrum Virtualize family storage systems

  • ibm_svc_mdiskgrp module – This module manages pools on IBM Spectrum Virtualize family storage systems

  • ibm_svc_start_stop_flashcopy module – This module starts or stops FlashCopy mapping and consistency groups on IBM Spectrum Virtualize family storage systems

  • ibm_svc_start_stop_replication module – This module starts or stops remote copies on IBM Spectrum Virtualize family storage systems

  • ibm_svc_vol_map module – This module manages volume mapping on IBM Spectrum Virtualize family storage systems

  • ibm_svcinfo_command module – This module implements SSH Client which helps to run svcinfo CLI command on IBM Spectrum Virtualize family storage systems

  • ibm_svctask_command module – This module implements SSH Client which helps to run svctask CLI command(s) on IBM Spectrum Virtualize family storage systems
