Community.Aws

Collection version 8.0.0

  • Description

  • Communication

  • Changelog

  • Plugin Index

Description

A variety of Ansible content to help automate the management of AWS services.

Author:

  • Ansible (https://github.com/ansible)

Supported ansible-core versions:

  • 2.15.0 or newer

  • Issue Tracker
  • Repository (Sources)

Communication

  • Matrix room #aws:ansible.im: General usage and support questions.

  • IRC channel #ansible-aws (Libera network): General usage and support questions.
