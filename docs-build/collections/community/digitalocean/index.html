Plugin Index

These are the plugins in the community.digitalocean collection:

Modules

  • digital_ocean module – Create/delete a droplet/SSH_key in DigitalOcean

  • digital_ocean_account_info module – Gather information about DigitalOcean User account

  • digital_ocean_balance_info module – Display DigitalOcean customer balance

  • digital_ocean_block_storage module – Create/destroy or attach/detach Block Storage volumes in DigitalOcean

  • digital_ocean_cdn_endpoints module – Create, update, and delete DigitalOcean CDN Endpoints

  • digital_ocean_cdn_endpoints_info module – Display DigitalOcean CDN Endpoints

  • digital_ocean_certificate module – Manage certificates in DigitalOcean

  • digital_ocean_certificate_info module – Gather information about DigitalOcean certificates

  • digital_ocean_database module – Create and delete a DigitalOcean database

  • digital_ocean_database_info module – Gather information about DigitalOcean databases

  • digital_ocean_domain module – Create/delete a DNS domain in DigitalOcean

  • digital_ocean_domain_info module – Gather information about DigitalOcean Domains

  • digital_ocean_domain_record module – Manage DigitalOcean domain records

  • digital_ocean_domain_record_info module – Gather information about DigitalOcean domain records

  • digital_ocean_droplet module – Create and delete a DigitalOcean droplet

  • digital_ocean_droplet_info module – Gather information about DigitalOcean Droplets

  • digital_ocean_firewall module – Manage cloud firewalls within DigitalOcean

  • digital_ocean_firewall_info module – Gather information about DigitalOcean firewalls

  • digital_ocean_floating_ip module – Manage DigitalOcean Floating IPs

  • digital_ocean_floating_ip_info module – DigitalOcean Floating IPs information

  • digital_ocean_image_info module – Gather information about DigitalOcean images

  • digital_ocean_kubernetes module – Create and delete a DigitalOcean Kubernetes cluster

  • digital_ocean_kubernetes_info module – Returns information about an existing DigitalOcean Kubernetes cluster

  • digital_ocean_load_balancer module – Manage DigitalOcean Load Balancers

  • digital_ocean_load_balancer_info module – Gather information about DigitalOcean load balancers

  • digital_ocean_monitoring_alerts module – Programmatically retrieve metrics as well as configure alert policies based on these metrics

  • digital_ocean_monitoring_alerts_info module – Programmatically retrieve metrics as well as configure alert policies based on these metrics

  • digital_ocean_project module – Manage a DigitalOcean project

  • digital_ocean_project_info module – Gather information about DigitalOcean Projects

  • digital_ocean_project_resource_info module – Gather information about DigitalOcean Project Resources

  • digital_ocean_region_info module – Gather information about DigitalOcean regions

  • digital_ocean_size_info module – Gather information about DigitalOcean Droplet sizes

  • digital_ocean_snapshot module – Create and delete DigitalOcean snapshots

  • digital_ocean_snapshot_info module – Gather information about DigitalOcean Snapshot

  • digital_ocean_spaces module – Create and remove DigitalOcean Spaces.

  • digital_ocean_spaces_info module – List DigitalOcean Spaces.

  • digital_ocean_sshkey module – Manage DigitalOcean SSH keys

  • digital_ocean_sshkey_info module – Gather information about DigitalOcean SSH keys

  • digital_ocean_tag module – Create and remove tag(s) to DigitalOcean resource.

  • digital_ocean_tag_info module – Gather information about DigitalOcean tags

  • digital_ocean_volume_info module – Gather information about DigitalOcean volumes

  • digital_ocean_vpc module – Create and delete DigitalOcean VPCs

  • digital_ocean_vpc_info module – Gather information about DigitalOcean VPCs
